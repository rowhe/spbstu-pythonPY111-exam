## Задание 1
## Оценить асимптотическую сложность приведенного ниже алгоритма:

```python
a = len(arr)–1 # Сложность O(1)
out = list() # O(1)
while a > 0: # O(n)
    out.append(arr[a])  # O(n)
    a = a // 1.7 # O(n)
out.merge_sort() # O(n log(n)
```

## Ответ:
### Алгоритм имеет общую асимптотическую сложность - O(n log(n))
